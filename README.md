# cnctd_npm

NPM wrapper for programmatic package management from Rust.

## Installation

```toml
[dependencies]
cnctd_npm = { git = "https://github.com/Connected-Dot/cnctd_crates.git" }
```

## Features

- Install/update dependencies
- Publish packages
- Run npm scripts
- Version management
